## desensitizeMobile

### API

```typescript
const data = desensitizeMobile(mobile: string | number);
```

### Params

| 参数 | 说明         | 类型     | 默认值 |
| ---- | ------------ | -------- | ------ |
| mobile | 入参，电话号码 | `string \| number` | -      |


### Result

| 参数 | 说明           | 类型     | 默认值 |
| ---- | -------------- | -------- | ------ |
| data | 返回的脱敏电话号 | `string` | ''     |
