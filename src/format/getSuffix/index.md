## getSuffix

获取文件名后缀

<code src="./demo.tsx"></code>

### API

```typescript
const data = getSuffix(name: string);
```

### Params

| 参数 | 说明         | 类型     | 默认值 |
| ---- | ------------ | -------- | ------ |
| name | 入参，文件名 | `string` | -      |


### Result

| 参数 | 说明     | 类型     | 默认值 |
| ---- | -------- | -------- | ------ |
| data | 返回的文件后缀 | `string` | ''      |
