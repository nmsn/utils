# usePrevious

获取变更前的参数

<code src="./demo.tsx"></code>

## API

```typescript
const prev = usePrevious(data: any);
```

### Params

| 参数 | 说明   | 类型  | 默认值 |
| ---- | ------ | ----- | ------ |
| data | 初始值 | `any` | -      |


### Result

| 参数 | 说明       | 类型  | 默认值 |
| ---- | ---------- | ----- | ------ |
| prev | 上一次的值 | `any` | -      |
