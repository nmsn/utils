## isAsyncFunction

判断参数是否为 `async function`

<code src="./demo.tsx"></code>

### API

```typescript
const flag: boolean = isAsyncFunction(param: unknown);
```

### Params

| 参数  | 说明 | 类型  | 默认值 |
| ----- | ---- | ----- | ------ |
| param | 参数 | `unknown` | -      |


### Result

| 参数 | 说明         | 类型      | 默认值  |
| ---- | ------------ | --------- | ------- |
| flag | 是否为 `async function` | `boolean` | `false` |
