## isMap

判断参数是否为 `Map`

<code src="./demo.tsx"></code>

### API

```typescript
const flag: boolean = isMap(param: unknown);
```

### Params

| 参数  | 说明 | 类型  | 默认值 |
| ----- | ---- | ----- | ------ |
| param | 参数 | `Map` | -      |


### Result

| 参数 | 说明         | 类型      | 默认值  |
| ---- | ------------ | --------- | ------- |
| flag | 是否是 'Map' | `boolean` | `false` |
