## isObj

判断参数是否为 `Object`

<code src="./demo.tsx"></code>

### API

```typescript
const flag: boolean = isObj(param: unknown);
```

### Params

| 参数  | 说明 | 类型      | 默认值 |
| ----- | ---- | --------- | ------ |
| param | 参数 | `unknown` | -      |


### Result

| 参数 | 说明       | 类型      | 默认值  |
| ---- | ---------- | --------- | ------- |
| flag | 是否是对象 | `boolean` | `false` |
