## isObj

判断参数是否为对象

<code src="./demo.tsx"></code>

### API

```typescript
const data = isObj(obj);
```

### Params

| 参数 | 说明 | 类型     | 默认值 |
| ---- | ---- | -------- | ------ |
| obj  | 参数 | `object` | -      |


### Result

| 参数 | 说明       | 类型      | 默认值 |
| ---- | ---------- | --------- | ------ |
| flag | 是否是对象 | `boolean` | -      |
