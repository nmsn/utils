## isValidArray

判断参数是否为有效的数组（非空数组）

<code src="./demo.tsx"></code>

### API

```typescript
const flag: boolean = isSameCharString(param: unknown);
```

### Params

| 参数  | 说明    | 类型      | 默认值 |
| ----- | ------- | --------- | ------ |
| param | 入參 | `unknown` | -      |


### Result

| 参数 | 说明           | 类型      | 默认值  |
| ---- | -------------- | --------- | ------- |
| flag | 是否为有效数组 | `boolean` | `false` |
