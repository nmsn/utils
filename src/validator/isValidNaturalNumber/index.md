## isValidNaturalNumber

判断参数是否为有效自然数

<code src="./demo.tsx"></code>

### API

```typescript
const flag: boolean = isValidNaturalNumber(num: number);
```

### Params

| 参数 | 说明 | 类型     | 默认值 |
| ---- | ---- | -------- | ------ |
| num  | 参数 | `number` | -      |


### Result

| 参数 | 说明             | 类型      | 默认值  |
| ---- | ---------------- | --------- | ------- |
| flag | 是否为有效自然数 | `boolean` | `false` |
