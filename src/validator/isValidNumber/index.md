## isValidNumber

判断参数是否为有效数字

<code src="./demo.tsx"></code>

### API

```typescript
const flag: boolean = isValidNumber(param: unknown);
```

### Params

| 参数 | 说明 | 类型     | 默认值 |
| ---- | ---- | -------- | ------ |
| param  | 参数 | `unknown` | -      |


### Result

| 参数 | 说明             | 类型      | 默认值  |
| ---- | ---------------- | --------- | ------- |
| flag | 是否为有效数字 | `boolean` | `false` |
