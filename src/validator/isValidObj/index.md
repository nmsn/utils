## isValidObj

判断参数是否有效的 `Object`（含有有效属性）

<code src="./demo.tsx"></code>

### API

```typescript
const flag: boolean = isValidObj(param: unknown);
```

### Params

| 参数  | 说明 | 类型      | 默认值 |
| ----- | ---- | --------- | ------ |
| param | 参数 | `unknown` | -      |


### Result

| 参数 | 说明       | 类型      | 默认值  |
| ---- | ---------- | --------- | ------- |
| flag | 是否为有效的 `Object` | `boolean` | `false` |
